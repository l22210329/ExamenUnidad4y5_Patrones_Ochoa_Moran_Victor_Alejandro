**Tema:** Agenda de recordatorios

---

## Descripción del Proyecto
El programa es una agenda de recordatorios con los patrones bridge, peso ligero, memento y MVVM.

### Funcionamiento
- **Build del programa dentro de la carpeta src:** `dotnet build -f net10.0-windows10.0.19041.0`
- **Ejecución:** Mediante consola con el comando `dotnet run -f net10.0-windows10.0.19041.0`
- **Salida:** Resultados en una interfaz.
